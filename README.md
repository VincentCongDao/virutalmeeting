## Run This Project Locally

Install the dependences

```bash
npm install
# or
yarn add
# or
bun add
```

```bash
npm run dev
# or
yarn run dev
# or
bun run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
